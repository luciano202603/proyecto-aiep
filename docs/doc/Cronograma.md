# Cronograma del Proyecto

| Día | Actividad |
|----|----------|
| Día 1 | Instalación del JDK y configuración del entorno |
| Día 2 | Desarrollo del código fuente en Java |

| Día 4 | Ejecución y pruebas del sistema |

| Día 7 | Subida de archivos y evidencias |
